## 문자열 압축

- 제일 앞부터 패턴 시작

- 1단위부터 n//2단위까지 모든 경우의 수를 탐색

- 뭐가 틀린걸까

  ```python
  result = result.replace('1', '')
  ```

  십의자리의 1도 빠져버림..

- 1인 경우 처리 안하니까 한 케이스에서 런타임 에러 발생

  ```
  if n == 1:
  	return 1
  ```
